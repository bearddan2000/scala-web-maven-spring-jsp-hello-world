# scala-web-maven-spring-jsp-hello-world

## Description
A springboot web app with jsp support.

## Tech stack
- scala
- maven

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`
Available at http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
